在服务器操作和日常编码中，掌握 Vim 的基础命令能够大幅提升我们的编辑效率。本文总结了 Vim 中最核心也是最常用的命令清单。

## 1. Vim 四种模式 {#vim-modes}

Vim 主要有四种常用的编辑模式：Normal 模式、命令模式、插入模式和可视化模式。

1. **Normal 模式**：通过按下 `ESC` 或 `Ctrl+[` 随时随地从其他模式返回至该模式，所有的快捷键指令都在此模式下触发。
2. **进入可视化模式（Visual Mode）**：
   - `v`：块状选择文本。
   - `V`：选择整行。
   - `Ctrl+V`：进行方块（列状）选择。

## 2. 插入模式 {#vim-insert}

在 Normal 模式下，可以使用如下命令快速切换进入**插入模式**：

```bash
i (insert) # 在当前光标处插入
I          # 在当前行首插入

a (append) # 在当前光标后开始插入
A          # 在当前行尾插入

o (open a line below) # 在光标下方新建一行开始插入
O (open a line up)    # 在光标上方新建一行开始插入
```

## 3. 快速编辑语法 {#vim-edit}

### 插入模式下的快速删除与切换

```bash
Ctrl+h # 删除上一个字符
Ctrl+w # 删除上一个单词
Ctrl+u # 删除当前行
Ctrl+[ # 代替 ESC 键，退回至 Normal 模式
```

### 单词与行删除 (Normal 模式)

```bash
daw # Delete Around Word: 删除一个单词（包含该单词周围的空格）
diw # Delete In Word: 删除单词内部（不包含两侧的空格）
d0  # 删除当前光标到行首的内容
d$  # 删除当前光标到行尾的内容
dd  # 删除整行
2d  # 向下删除2行
2x  # 删除光标后的2个字符
```

> `d` (删除) 和 `x` (剪切删除) 可以配合 **Visual 可视化模式** 来批量消除选中的文本。

### 内容修改与替换

主要使用 `c` (change)、`r` (replace) 和 `s` (substitute) 命令。

```bash
r   # (replace) 仅替换光标所在的那一个字符
R   # 进入连续替换模式
c   # (change) 修改操作，例如：caw、ct) 修改从光标处直到括号内的元素
s   # (substitute) 删除当前字符并进入插入模式
S   # 删除整行并进入插入模式
```

### 快速查找

```bash
/   # 向下进行寻找
?   # 向上进行寻找
n/N # 跳到上一个/下一个匹配的结果
```

## 4. 快速移动光标 {#vim-movement}

放弃方向键，让双手不离键盘主区。

### 基础与跳转移动

```bash
h、j、k、l # 分别对应光标的 左、下、上、右

gi # 快速跳转到你最后一次修改编辑的地方，并自动进入插入模式

w/W # 下一个单词开头
e/E # 下一个单词结尾
b/B # 上一个单词开头
# 提示：小写字母以“非空白符”分割，大写字母严格以“空白符”分割单词区块。
```

### 单行内的按字符跳转

```bash
f{char} # 正向移动到该行中下一个匹配为 char 的字符上
t{char} # 正向移动到该行中下一个匹配为 char 字符的前一个位置
# 提示：大写的 F 和 T 命令表示“反向往前找”。同时配合 `;` (分号) 和 `,` (逗号) 可以继续搜索该行后续/前面的结果。
```

### 行内、行首与跳跃

```bash
0  # 移动到绝对行首
^  # 移动到第一个非空白字符
$  # 移动到绝对行尾
g_ # 移动到行尾的最后一个非空白字符

3$ # 跳转并移动到当前行往下数的第3行行尾
```

### 屏幕与全局文件跳转

```bash
gg / G        # 移动到文件绝对开头 / 移动到文件绝对结尾
Ctrl+o        # 快速返回上一次跳转前的位置

H / M / L     # 跳转到当前屏幕状态的顶部 (Head) / 中间 (Middle) / 底部 (Low)

Ctrl+u/Ctrl+f # 向上/向下翻页滚动
zz            # 把当前行滚动到屏幕的中间位置
```

## 5. 复制、粘贴与剪贴 {#vim-copy-paste}

灵活运用这几组命令，彻底摆脱鼠标操作。

```bash
y (yank)   # 复制
p (paste)  # 粘贴
d          # 剪贴删除（被 dd 或 daw 删除的内容可以被 p 命令重新粘贴）

yiw        # 复制一个光标所停靠的单词（Yank in word）
yy         # 复制当前的整行
xp         # 由于 x 是剪掉当前字符，p 是在光标后粘贴，两者连用即相当于互换当前和下一个字符顺序！
```
